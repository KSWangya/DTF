## 📖 專案簡介

這是**岡山大埕代天府**的官方網站，採用現代化網頁設計結合傳統東方美學，展現百年古廟的莊嚴與信仰精神。

## ✨ 網站特色

- 🎨 **新東方美學設計** - 米色宣紙底、絳紅印泥、沉金配色
- 🏯 **直排書法標題** - 傳統書法排版風格
- ✨ **香火粒子動畫** - Canvas 繪製的飄動香火效果
- ☁️ **浮動雲朵** - 增添廟宇神聖氛圍
- 📱 **響應式設計** - 完美支援桌面與行動裝置
- 🎞️ **Lightbox 圖庫** - 點擊放大瀏覽廟宇影像
- 🔄 **平滑滾動動畫** - 視差效果與進場動畫

## 🛠️ 技術棧

- **HTML5** - 語義化結構
- **CSS3** - Flexbox / Grid 佈局、CSS 動畫、漸層效果
- **Vanilla JavaScript** - 原生 JS，無框架依賴
- **Canvas API** - 香火粒子效果
- **Intersection Observer API** - 滾動觸發動畫
- **Google Fonts** - Noto Serif TC / Noto Sans TC
- **Font Awesome** - 圖示字型

## 🚀 使用方式

1. 直接使用瀏覽器開啟 `index.html`
2. 或使用本地伺服器：

   ```bash
   # 使用 Python
   python -m http.server 8080

   # 或使用 Node.js
   npx serve .
   ```

3. 開啟瀏覽器訪問 `http://localhost:8080`
